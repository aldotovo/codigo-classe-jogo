# Heróis de Aventura

Este projeto permite criar heróis 👷 com:

- **Nome** 🌟
- **Idade** ⏳
- **Tipo** (ex: mago, guerreiro, monge, ninja) ⚔️

### Ataques personalizados ✨

Cada tipo de herói tem um ataque exclusivo:

- **Mago**: magia 🌌
- **Guerreiro**: espada ⚔️
- **Monge**: artes marciais 🏃
- **Ninja**: shuriken 🎮
